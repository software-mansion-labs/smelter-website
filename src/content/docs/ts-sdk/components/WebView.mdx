`WebView` renders a website using Chromium engine embedded inside the Smelter instance.

<Aside>To use this component, you need to first register the image with a matching `instanceId` using the `Smelter.registerWebRenderer` function.</Aside>

<Aside type='caution'>Only one component can use specific `instanceId` at the time.</Aside>

## Props

### instanceId
ID of a web renderer instance. It identifies an instance registered using a [Smelter.registerWebRenderer](/ts-sdk/smelter#registerwebrenderer) method.

- **Type**: `string`

---

### children
Content to be displayed within the `WebView`.

- **Type**: `ReactNode`

---

### id
ID of a component.

- **Type**: `string`
- **Default value**: Value produced by `useId` hook
